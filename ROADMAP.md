# ROADMAP

## Mission
Build an AI-native open-source project that is maintained continuously over time, not just generated once.

## Phase 1 — Bootstrap
- [x] Define project direction
- [x] Add transparent AI disclosure
- [x] Ship a minimal working CLI
- [ ] Push first public commit

## Phase 2 — Repository intelligence
- [ ] Read git history and summarize change patterns
- [ ] Detect stale files and maintenance risks
- [ ] Output JSON schema for downstream agent use

## Phase 3 — Hosting integrations
- [ ] Optional GitHub issue ingestion
- [ ] Optional PR / release draft generation
- [ ] Scheduled maintenance workflows

## Phase 4 — AI maintenance loop
- [ ] Suggest next tasks from repository state
- [ ] Generate human-reviewable weekly plans
- [ ] Track long-term repository health over time
